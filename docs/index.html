cad-agent v0.2.0

AI parametric CAD用一句話，建出一個 3D 零件

描述零件，或拍一張實物照，AI 寫出參數化 FreeCAD 腳本、無頭建置，瀏覽器即時看結果。尺寸就是參數：改數字，幾秒重建，不用再等 AI。

GitHub 原始碼 本機安裝

特色 FEATURES

建置進度全程可見

AI 寫腳本中、FreeCAD 建置中、自我修復第 N 次，狀態即時串流到瀏覽器。

參數面板

AI 寫的每個尺寸都是可編輯欄位，改數字直接重跑 FreeCAD，不經 AI、秒級完成。

STL/STEP 下載

CAD 工具的產出就是檔案，建完直接拿去印或進 CAD 軟體。

建置歷史

每次成功建置是一個版次（R1、R2…），可回復任一版繼續改。

拍照逆向建模

上傳實物照片，AI 逆向出參數化模型；附一個已知尺寸就能定比例。

失敗自我修復

建置錯誤自動回灌給 AI 修正，最多重試兩次。

本機安裝 INSTALL

git clone https://github.com/yazelin/cad-agent && cd cad-agent
python -m venv .venv && . .venv/bin/activate
pip install -e ".[dev]"
python -m cad_agent   # 開 http://127.0.0.1:8099

需要：Python 3.11+、FreeCAD(headless)、已登入的 claude CLI（用你的訂閱，不走 API 計費）。這是在你自己機器上跑的工具，這一頁只是展示。
